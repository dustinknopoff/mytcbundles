### Constant Variable

Description:

> Creates constant variable.

` const int size 300`

will render:


```java
 private static final int SIZE = 300;
 
```

Languages: ['java']



### If

Description:

> Creates if statement.

` if x > 300`

will render:


```java
 if (x > 300) {
     
 }
```

Languages: ['java', 'js', 'c']



### For

Description:

> Creates for loop.

` for int ArrayList arr`

will render:


```java
 for (int i : ArrayList arr) {
     
 }
```

Languages: ['java']



### For(i)

Description:

> Creates iterative for loop.

` fori <= arr.length()`

will render:


```java
 for (int i = 0; i <= arr.length(); i++) {
     
 }
```

Languages: ['java', 'js', 'c']


### Interface

Description:

> Creates an interface in Java.

` i Car im base`

will render:


```java

 }
```

will render:


```java

 }
```
